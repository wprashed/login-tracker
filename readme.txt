The Login Tracker plugin adds a dashboard widget to the WordPress admin dashboard that displays a user's login data, including the date and IP address of each login. This information can be useful for tracking login activity and identifying any potential security issues.

== Description ==

With this plugin, you can easily view the login history for your user account and stay informed about any unusual login activity. Whether you are an individual user or an administrator managing multiple accounts, the Login Tracker can help you keep track of your login activity and ensure the security of your WordPress site.

The plugin is easy to use and integrates seamlessly with the WordPress dashboard. Simply install and activate the plugin, and the login data widget will be automatically added to the dashboard. No additional configuration is needed.

Overall, the Login Tracker is a valuable tool for any WordPress user looking to track and monitor their login activity.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `login-tracker.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
